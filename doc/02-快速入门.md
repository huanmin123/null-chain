# 快速入门

## 环境要求

- **JDK 版本**：JDK 8~25 及以上
- **构建工具**：Maven 或 Gradle

## 添加依赖

### Maven 依赖

```xml
<!-- 核心模块（必需） -->
<dependency>
    <groupId>com.gitee.huanminabc</groupId>
    <artifactId>null-chain-core</artifactId>
    <version>最新版本</version>
</dependency>
```

### Gradle 依赖

```gradle
// 核心模块（必需）
implementation 'com.gitee.huanminabc:null-chain-core:最新版本'

// Spring Boot 集成（可选）
implementation 'com.gitee.huanminabc:null-chain-boot-starter:最新版本'
```

## 基本使用

### 1. 创建 Null 链

```java
import com.gitee.huanminabc.nullchain.Null;
import com.gitee.huanminabc.nullchain.core.NullChain;

// 创建包含值的链
NullChain<String> chain = Null.of("Hello");

// 创建空链
NullChain<String> emptyChain = Null.empty();

// 从可能为 null 的值创建链
User user = getUserById(1);
NullChain<User> userChain = Null.of(user);
```

### 2. 基本操作

#### 映射操作

```java
// 映射操作
String upperName = Null.of(user)
    .map(User::getName)
    .map(String::toUpperCase)
    .orElse("UNKNOWN");
```

#### 条件判断

```java
// 条件判断
String result = Null.of(user)
    .ifGo(u -> u.getAge() > 18)
    .map(User::getName)
    .orElse("未成年用户");
```

#### 获取值

```java
// 获取值
String name = Null.of(user)
    .map(User::getName)
    .orElse("默认名称");
```

#### 检查是否为空

```java
// 检查是否为空
boolean isEmpty = Null.of(user)
    .map(User::getName)
    .is();  // 返回 true 表示值为空
```

### 3. 集合操作

```java
// 从集合创建流
List<String> names = Null.ofStream(users)
    .map(User::getName)
    .filter(Objects::nonNull)
    .collect(Collectors.toList());

// 从 Stream 创建
Stream<String> stream = Stream.of("a", "b", "c");
List<String> list = Null.ofStream(stream)
    .map(String::toUpperCase)
    .collect(Collectors.toList());
```

### 4. 多级对象访问

```java
// 多级映射
String roleName = Null.of(user)
    .map(User::getRole)
    .map(Role::getName)
    .orElse("无角色");
```

### 5. 条件分支

```java
// 如果用户名为空，链变为空链
String result = Null.of(user)
    .of(User::getName)  // 如果用户名为空，链变为空链
    .map(String::toUpperCase)
    .orElse("UNKNOWN");
```

### 6. 执行操作但不改变值

```java
// 执行操作但不改变值
Null.of(user)
    .then(() -> log.info("处理用户数据"))
    .map(User::getName)
    .orElse("未知用户");

// 带参数的操作
Null.of(user)
    .peek(u -> log.info("处理用户: {}", u.getName()))
    .map(User::getName)
    .orElse("未知用户");
```

## 完整示例

```java
import com.gitee.huanminabc.nullchain.Null;
import java.util.List;
import java.util.stream.Collectors;

public class Example {
    
    public void processUser(User user) {
        // 获取用户名并转为大写
        String upperName = Null.of(user)
            .map(User::getName)
            .map(String::toUpperCase)
            .orElse("UNKNOWN");
        
        // 条件判断
        String message = Null.of(user)
            .ifGo(u -> u.getAge() > 18)
            .map(User::getName)
            .orElse("未成年用户");
        
        // 多级对象访问
        String roleName = Null.of(user)
            .map(User::getRole)
            .map(Role::getName)
            .orElse("无角色");
    }
    
    public void processUsers(List<User> users) {
        // 集合操作
        List<String> names = Null.ofStream(users)
            .map(User::getName)
            .filter(Objects::nonNull)
            .collect(Collectors.toList());
    }
}
```

## 问题(缺少依赖 或者 版本问题)
大于 1.8 版本的因为有模块限制需要开放一些权限
```xml

    <dependencies>
        <dependency>
            <groupId>com.gitee.huanminabc</groupId>
            <artifactId>null-chain-core</artifactId>
            <version>最新版本</version>
        </dependency>

        <dependency>
            <groupId>com.gitee.huanminabc</groupId>
            <artifactId>null-chain-work</artifactId>
            <version>最新版本</version>
        </dependency>

        <dependency>
            <groupId>org.junit.jupiter</groupId>
            <artifactId>junit-jupiter</artifactId>
            <version>5.7.2</version>
        </dependency>

        <dependency>
            <groupId>org.projectlombok</groupId>
            <artifactId>lombok</artifactId>
            <version>1.18.42</version>
        </dependency>

        <dependency>
            <groupId>ch.qos.logback</groupId>
            <artifactId>logback-core</artifactId>
            <version>1.2.7</version>
        </dependency>

    </dependencies>

    <build>
        <plugins>
            <plugin>
                <groupId>org.apache.maven.plugins</groupId>
                <artifactId>maven-surefire-plugin</artifactId>
                <configuration>
                    <argLine>
                        --add-opens java.base/java.lang=ALL-UNNAMED
                        --add-opens java.base/java.util=ALL-UNNAMED
                        --add-opens java.base/java.lang.invoke=ALL-UNNAMED</argLine>
                </configuration>
            </plugin>

        </plugins>
    </build>
```
